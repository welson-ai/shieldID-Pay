# ShadowID Pay

A decentralized payment system built with compact smart contracts and a React frontend.

## Project Structure

```
shieldid-pay/
├── contracts/          # Compact smart contracts
├── src/               # React frontend
├── tests/             # Contract tests
└── README.md
```

## Overview

ShadowID Pay is a minimal yet powerful payment system that leverages blockchain technology for secure, transparent transactions.

### Components

- **Contracts**: Efficient smart contracts for payment processing
- **Frontend**: React-based user interface for interacting with the payment system
- **Tests**: Comprehensive test suite for contract validation
